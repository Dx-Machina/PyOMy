# Project: PyOMy
# Description: AI-voice assistant fused with OpenAI's ChatGPT. It produces voice and written responses.
# Hardware Requirements:
	Raspberry Pi4(8gb),
	Microphone,
	Speakers,
	Keyboard.
	Mouse.
# Software:
	OpenAI Account (free tier will suffice),
	RPiOS (64-bit),
	Python 3.6 or Later,
	Linux Terminal,
	pip,
	bash,
	git,
	nano.
